### Changed

- Modernized the runtime layout around a Node 24 entrypoint and container image.
- Added built-in Node test coverage for app routes, configuration parsing, and metric ingestion.
- Added GitHub Actions workflows for CI, PR containers, release management, and release containers.
- Added repository policy and maintenance files including AI notice, changelog, contributing, security, and Renovate configuration.
